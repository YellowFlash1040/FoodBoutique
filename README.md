## Usage

<p>To try out this web grocery shop you need to make a few simple steps : </p>
<ol style="display: flex; flex-direction: column; gap: 20px;">
  <li>
    <p>Navigate to the <a href="https://yellowflash1040.github.io/FoodBoutique" target="_blank">https://yellowflash1040.github.io/FoodBoutique</a> page
    </p>
    <div align="center">
    <img src="./src/public/README assets/home-page.png" alt="Expense Tracker homepage" height="467"/>
    </div>
  </li>
  <li>
    <p>Add all the products you like to the shopping cart by just clicking on the cart icon next to them
    </p>
    <div align="center">
    <img src="./src/public/README assets/add-product-action.png" alt="Product adding process" height="300"/>
    </div>
    Or you can click on the product itself and you will see a popup with the detailed product info. You can also add it to the cart from here
    <div align="center">
    <img src="./src/public/README assets/product-info-popup.png" alt="Detailed product info" height="300"/>
    </div>
    You can also filter products using filters panel
    <div align="center">
    <img src="./src/public/README assets/filters.png" alt="Filters usage" height="300"/>
    </div>
  </li>
  <li>
    <p>After all the products have been added, you need to return to the very top of the page and find the cart icon in the upper right corner
    </p>
    <div align="center">
    <img src="./src/public/README assets/cart-page-button.png" alt="Cart page button" />
    </div>
  </li>
  <li>
    <p>Once you clicked this button you will be redirected to the Cart page where you can view a list of all added products and their total cost
    </p>
    <div align="center">
    <img src="./src/public/README assets/cart-page.png" alt="Food Boutique cart page" width="890"/>
    </div>
  </li>
  <li>
    <p>Here you can find a form where you can enter an email where the payment confirmation will be sent
    </p>
    <div align="center">
    <img src="./src/public/README assets/payment-confirmation-form.png" alt="Payment confirmation form" />
    </div>
    <p>Next you can click "Checkout" button to proceed to checkout
    </p>
  </li>
  <li>
    <p>Once you clicked it the order will be sent and the confirmation popup will be displayed
    </p>
    <div align="center">
    <img src="./src/public/README assets/order-confirmation-popup.png" alt="Order confirmation popup" />
    </div>
  </li>
  <li>
    <p>Also if you want to you can subscribe to our news letters using form at the very page bottom
    </p>
    <div align="center">
    <img src="./src/public/README assets/subscribe-form.png" alt="News letters subscribe form" />
    </div>
  </li>
</ol>

## For developers

If you want to expand or improve this project, or you just want to run it
locally you can follow this steps :

<ol>
  <li>Clone this repository to your local machine</li>
  <li>Make sure you have Node.js installed on your local machine</li>
  <li>If not than install it from the official website <a href="https://nodejs.org/en/download">https://nodejs.org/en/download</a></li>
  <li>Run <code>npm install</code> command to install all of the dependencies</li>
  <li>Run <code>npm run dev</code> command to run the app in the developer mode</li>
  <li>Create a separate git branch</li>
  <li>Make changes that you want to</li>
  <li>Make a commit</li>
  <li>Push it to a remote branch</li>
  <li>Create a pull request</li>
  <li>Wait untill I will approve or decline the changes</li>
</ol>
